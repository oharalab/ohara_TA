【ポインタ変数を利用した文字列の表示】

文字列を保存した char型の配列をアドレス渡しで受け取り，その内容を1文字ずつ for文を
使って表示する関数 print_string() を実装し，以下のプログラム ex11_1.c を完成させよ．
ただし，関数 print_string() 内では，if 文を利用してはならず，また，配列も利用しては
ならない．文字列の表示には char型のポインタ変数を利用すること．
なお，ex11_1.c は以下の部分に各関数の定義を加えるのみとし，以下に示している
部分に関しては，一切の変更・追記は認めない．

#include<stdio.h>
#include<stdlib.h>

#define BUFSIZE 64

void print_string(char *s);

int main() {
    char string[BUFSIZE] = "Aoyama Taro";
	
	print_string(string);
	
    return 0;
}


実行結果例（赤文字がキーボードからの入力）：
$ ./ex11_1.exe
Aoyama Taro
