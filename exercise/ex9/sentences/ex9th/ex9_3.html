実習課題9_3

【九九の表の出力】

実習課題9_2 で実装した関数 isInt() を用いて，キーボードから入力された1から9までの整数値 n に対して，
n x n までの九九の表を出力するプログラム ex9_3.c を作成せよ．ex9_3.c における関数 print_kuku() が
コンソール上に九九の表を出力するものとする．
なお，ex9_3.c は以下の部分に関数 isInt()，getInput()，print_kuku() の定義を加えるのみとし，以下に示して
いる部分に関しては，一切の変更・追記は認めない．
以下の部分が変更・追記されたプログラムは正しく動作しても 0 点とする．

#include<stdio.h>
#include<ctype.h>   // 数字の判定に利用
#include<string.h>  // 文字列長の取得に利用
#include<stdlib.h>  // atoi() の利用

#define BUFSIZE 128

void getInput(char msg[], char n[]);
int isInt(char n[]);
void print_kuku(int n);

int main(){
    char buf[BUFSIZE];
    int n;
	
    getInput("1から9までの整数値を入力してください：", buf);
    while(1){
        if(!isInt(buf))
            printf("%s は整数値ではありません\n", buf);
        else if(0 < (n = atoi(buf)) && n <= 9)
            break;
        getInput("1から9までの整数値を入力してください：", buf);
    }
	print_kuku(n);
	
    return 0;
}

実行例1（赤文字がキーボードからの入力）：

$ ./ex9_3.exe
1から9までの整数値を入力してください：0
1から9までの整数値を入力してください：10
1から9までの整数値を入力してください：a
a は整数値ではありません
1から9までの整数値を入力してください：3
     1  2  3
  1  1  2  3
  2  2  4  6
  3  3  6  9
