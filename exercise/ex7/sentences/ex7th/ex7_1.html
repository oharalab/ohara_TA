【データ入力関数の作成】

演習7-1，7-6 を参考に，演習7-9 で作成した p7_9.c を，データの入力をキーボードから
できるように変更したプログラム ex7_1.c を作成せよ．
キーボードからの入力は，関数 get_data() により実現するものとする．
具体的には，以下のコードに対して，関数 get_data() と 関数 print_data() を
実装することで ex7_1.c を完成させよ．
なお，関数 get_data() と 関数 print_data() の実装以外に，以下に示しているコードを
変更してはならない（何かを追記することも不可）．

#include<stdio.h>
#include<string.h>
#include<stdlib.h>

#define MAXNUM 10
#define BUFSIZE 256

typedef struct {
    char name[30];
    int age;
    int grade;
    double gpa;
} personal_data;

void print_data(personal_data d[]);
personal_data get_data(void);

int main() {
    int i;
    personal_data data[MAXNUM];
	
    for(i = 0; i < MAXNUM; i++){
        data[i] = get_data();
        if(!strcmp(data[i].name, "end")) break;
    }

    print_data(data);
	
    return 0;
}
