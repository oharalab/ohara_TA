【指定した年月日の曜日の表示】

実習課題8_14 作成した ex8_14.c に，指定した年の1月1日の曜日と，その年における指定した日付までの
日数を手掛かりに，その年月日の曜日を求める関数 getDayOfWeek() を実装し，以下のプログラム ex8_15.c
を完成させよ．関数 getDayOfWeek() の第1引数は指定した年の1月1日の曜日に対応するインデックス，第2引数は
その年における指定した日付までの日数である．
なお，ex8_15.c は以下の部分に各関数の定義を加えるのみとし，以下に示している部分に
関しては，一切の変更・追記は認めない．
以下の部分が変更・追記されたプログラムは正しく動作しても 0 点とする．

#include<stdio.h>
#include<ctype.h>
#include<string.h>
#include<stdlib.h>

#define BUFSIZE 128

int days[] = { 31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31 };
char day_of_week[7][10]={
	"sunday","monday","tuesday", "wednesday", "thursday","friday", "saturday"};

void getInput(char msg[], char n[]);
int isInt(char n[]);
int getYear(void);
int getMonth(void);
int getDay(int y, int m);
int getDays(int y, int m);
int isLeapYear(int y);
int getDayOfWeekForFirstDay(int y);
int countDays(int y, int m, int d);
int getDayOfWeek(int dow, int days);

int main(){
	int year, month, day, dow, dow_first, days_in_year;
	
	year = getYear();
	printf("西暦：%d\n", year);
	month = getMonth();
	printf("月：%d\n", month);
	day = getDay(year, month);
	printf("日：%d\n", day);
	dow_first = getDayOfWeekForFirstDay(year);
	printf("%d年の1月1日の曜日は %s\n", year, day_of_week[dow_first]);
	days_in_year = countDays(year, month, day);
	printf("%d年%d月%d日はその年の%d日目\n", year, month, day, days_in_year);
	dow = getDayOfWeek(dow_first, days_in_year);
	printf("曜日は %s です\n", day_of_week[dow]);

	return 0;
}


実行例（赤文字がキーボードからの入力）：
