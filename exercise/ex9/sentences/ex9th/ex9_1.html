【非負整数の入力チェック】

非負整数（0以上の整数）として入力した文字列が標準ライブラリ関数 atoi() で正しく意図した数値に変換できる
ことを目的として，入力した文字列が非負整数（0以上の整数）であることを正しく判定する関数 isInt() を実装し，
以下のプログラムを ex9_1.c として完成させよ．ここでの非負整数は，小数点を伴ってもよいものとするが，小数点の
前には1桁以上の数値が存在する必要があるものとする．
関数 isInt() は，引数として与えられた文字列が非負整数の場合は 1 を返し，それ以外の場合は 0 を返すものとする．
また，関数 getInput() は，先週の sample8_1.c の関数 print_msg() を参考に実装すること．
なお，ex9_1.c は以下の部分に関数 isInt() と getInput() の定義を加えるのみとし，以下に示している部分に
関しては，一切の変更・追記は認めない．
以下の部分が変更・追記されたプログラムは正しく動作しても 0 点とする．

#include<stdio.h>
#include<ctype.h>   // 数字の判定に利用
#include<string.h>  // 文字列長の取得に利用

#define BUFSIZE 128

void getInput(char msg[], char n[]);
int isInt(char n[]);

int main(){
    char buf[BUFSIZE];
	
    getInput("整数値を入力してください：", buf);
    if(isInt(buf))
        printf("%s は非負の整数値です\n", buf);
    else
        printf("%s は非負の整数値ではありません\n", buf);
	
    return 0;
}


実行例1（赤文字がキーボードからの入力）：

$ ./ex9_1.exe
整数値を入力してください：123
123 は非負の整数値です


実行例2（赤文字がキーボードからの入力）：
