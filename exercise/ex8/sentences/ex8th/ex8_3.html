実習課題8_3

【累乗値】

0から19までの2乗値・3乗値をそれぞれ配列に格納した後，それらを表示せよ．
手順は以下の通り．

  • あらかじめ，2乗値を格納する配列，および3乗値を格納する配列を用意する．
  • for文を使用し，それぞれの計算値を各配列に格納する．
  • 再び，for文を用いて，配列の中身を出力する．
配列を使用しなかった場合は 0 点.
なお，プロジェクト名は ex8_3とし，ソースファイルの名前は ex8_3.c とする．

＜ヒント＞
a の2乗は a x a，3乗は a x a x a ．
今回の課題では2乗値，3乗値を求めるために第3回の授業で学習した
pow(a, b) は使うべきではない．これは，配列のインデックスは int 型（整数）
であるのに対し，pow(a, b) の引数 a，b は共に double 型であるため．
