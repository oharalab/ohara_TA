実習課題9_4

【九九の表の出力（ファイルへの出力を含む）】

実習課題9_3b で作成した ex9_3.c に，九九の表をファイルに出力する関数 print_kukuf() をさらに
実装したプログラム ex9_4.c を作成せよ．出力ファイル名は "result_n.txt" とし，n の部分は
キーボードから入力された数値となるものとする．
また，ex9_3.c では main() 関数内に記述されていたキーボードからの入力を受け取る部分は
関数 getValue() として実装せよ．
なお，ex9_4.c は以下の部分に各関数の定義を加えるのみとし，以下に示している部分に関しては，
一切の変更・追記は認めない．
以下の部分が変更・追記されたプログラムは正しく動作しても 0 点とする．

#include<stdio.h>
#include<ctype.h>   // 数字の判定に利用
#include<string.h>  // 文字列処理に利用
#include<stdlib.h>  // atoi() とエラー処理に利用

#define BUFSIZE 128

void getInput(char msg[], char n[]);
int isInt(char n[]);
void print_kuku(int n);
void print_kukuf(int n, char buf[]);
int getValue(char buf[]);

int main(){
    char buf[BUFSIZE];
    int n;
	
    n = getValue(buf);
    print_kuku(n);
    print_kukuf(n, buf);
	
    return 0;
}

実行例1（赤文字がキーボードからの入力）：

$ ./ex9_4.exe

a は整数値ではありません
