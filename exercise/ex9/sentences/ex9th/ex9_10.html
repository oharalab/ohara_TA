【西暦・月・日の入力】

実習課題9_9 で作成した ex9_9.c に，指定した年月に対する正しい日の入力を求める関数 getDay()
を実装し，以下のプログラム ex9_10.c を完成させよ．
関数 isLeapYear() は引数で与えられた西暦年がうるう年であれば 1 を，そうでなければ 0 を返す
関数であり，関数 getDays() は引数で与えられた年月の日数を返す関数である．
これらも，今週までの課題を参考に実装すること．
なお，ex9_10.c は以下の部分に各関数の定義を加えるのみとし，以下に示している部分に
関しては，一切の変更・追記は認めない．
以下の部分が変更・追記されたプログラムは正しく動作しても 0 点とする．

#include<stdio.h>
#include<ctype.h>
#include<string.h>
#include<stdlib.h>

#define BUFSIZE 128

int days[] = { 31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31 };

void getInput(char msg[], char n[]);
int isInt(char n[]);
int getYear(void);
int getMonth(void);
int getDay(int y, int m);
int getDays(int y, int m);
int isLeapYear(int y);

int main(){
	int year, month, day;
	
	year = getYear();
	printf("西暦：%d\n", year);
	month = getMonth();
	printf("月：%d\n", month);
	day = getDay(year, month);
	printf("日：%d\n", day);

	return 0;
}
