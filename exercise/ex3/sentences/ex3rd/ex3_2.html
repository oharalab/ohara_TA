実習課題3_2

以下のプログラムには複数の間違いが含まれている．
実行例と同じ結果が得られるようにデバッグせよ．
ただし，printf() では，画面に出力する数値を直接書いてはならない．
プロジェクト名は ex3_2 とし，ソースファイルの名前は ex3_2.c とする．

手順としては，まず以下のコードを ex3_2.c として作成し，その上でデバッグすること． 

#include <stdio.h>

int main(){
	double a, b, c;

	a = log10(c);
	b = log10(6.0 / 16.0);

	printf("a = %d\n", a);
	printf("b = %f\n", b);
	printf("c = %d\n", c);

	printf("b = %f\n", log10(2) + log10(3) - log10(16));

	return(0);
}

実行例 

a =   2.000
b =  -0.426
c = 100.000
b =  -0.426
