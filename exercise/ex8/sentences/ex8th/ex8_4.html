実習課題8_4

【整数の入力チェック】

実習課題8_3 で実装した関数 isInt() を負の整数も含む整数値を判定するように変更したプログラム ex8_4.c を
作成せよ．ex8_4.c における関数 isInt() は，引数として与えられた文字列が整数の場合は 1 を返し，それ以外の場合は
0 を返すものとする．
なお，ex8_4.c は以下の部分に関数 isInt() と getInput() の定義を加えるのみとし，以下に示している部分に
関しては，一切の変更・追記は認めない．
以下の部分が変更・追記されたプログラムは正しく動作しても 0 点とする．

#include<stdio.h>
#include<ctype.h>   // 数字の判定に利用
#include<string.h>  // 文字列長の取得に利用

#define BUFSIZE 128

void getInput(char msg[], char n[]);
int isInt(char n[]);

int main(){
    char buf[BUFSIZE];
	
    getInput("整数値を入力してください：", buf);
    if(isInt(buf))
        printf("%s は整数値です\n", buf);
    else
        printf("%s は整数値ではありません\n", buf);
	
    return 0;
}

実行例1（赤文字がキーボードからの入力）：

$ ./ex8_4
整数値を入力してください：123
123 は整数値です


実行例2（赤文字がキーボードからの入力）：

$ ./ex8_4
整数値を入力してください：12.3
12.3 は整数値ではありません


実行例3（赤文字がキーボードからの入力）：

$ ./ex8_4
整数値を入力してください：0
0 は整数値です


実行例4（赤文字がキーボードからの入力）：

実行例5（赤文字がキーボードからの入力）：

実行例6（赤文字がキーボードからの入力）：

実行例7（赤文字がキーボードからの入力）：
