実習課題7_3

【構造体配列を利用するデータ入力関数の作成】

実習課題7_1 で作成した ex7_1.c は，main() 関数内にデータ入力に関する for文が残っているため
プログラムの構造化という点では不十分である．
実習課題7_2 における観察結果を参考に，ex7_1.c 中の main() 関数にある for文を関数 get_data()内に
移したプログラム ex7_3.c を作成せよ．
具体的には，実習課題7_1 と同様に，以下のコードに対して関数 print_data() と関数 get_data() を実装する
ことで，プログラム ex7_3.c を完成させよ．ここで，関数 print_data() は ex7_1.c と同じものを想定している．
なお，実習課題7_1 と同様に，関数 print_data() と関数 get_data() を実装する以外は，
以下のコードを変更してはならない（何かを追記することも不可）．

#include<stdio.h>
#include<string.h>
#include<stdlib.h>

#define MAXNUM 10
#define BUFSIZE 256

typedef struct {
    char name[30];
    int age;
    int grade;
    double gpa;
} personal_data;

void print_data(personal_data d[]);
void get_data(personal_data d[]);

int main() {
    int i;
    personal_data data[MAXNUM];
	
    get_data(data);
    print_data(data);
	
    return 0;
}
