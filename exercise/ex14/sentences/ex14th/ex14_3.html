【カンマ区切りの文字列入力】

p14_3.c を参考に，カンマで区切った複数の文字列を読み込むプログラム ex14_3.c を作成せよ．
文字列数は任意とし，読み込んだ文字列は線形リストで実現したスタックで保持するものとする．
なお，ex14_3.c は以下の部分に各関数の定義を加えるのみとし，以下に示している部分に
関しては，一切の変更・追記は認めない．
以下の部分が変更・追記されたプログラムは正しく動作しても 0 点とする．

#include<stdio.h>
#include<stdlib.h>;
#include<string.h>

typedef struct stack{
	char *string;
	struct stack *next;
} stack_node;

void remove_nl(char *s);
stack_node *get_strings(char *buf);
stack_node *new_node(char *s);
stack_node *push_node(stack_node *root, char *s);
void print_stack(stack_node *np);


int main(){
	char buf[1024];
	stack_node *root;
	
	printf("カンマで区切った文字列を入力してください：");
	fgets(buf, sizeof(buf), stdin);
	remove_nl(buf);
	root = get_strings(buf);
	print_stack(root);

	return 0;
}

ファイルの先頭には学生番号と氏名を必ず明記すること．
