実習課題8_3

【非負整数の入力チェック】

実習課題8_1 と 8_2 の観察結果を踏まえて，非負の整数値（0 以上の整数値）の入力が期待されるときに，
入力が非負整数であることを正しく判定する関数 isInt() を実装し，以下のプログラムを ex8_3.c として
完成させよ．関数 isInt() は，引数として与えられた文字列が非負整数の場合は 1 を返し，それ以外の場合は
0 を返すものとする．また，関数 getInput() は，sample8_1.c の関数 print_msg() を参考に実装すること．
なお，ex8_3.c は以下の部分に関数 isInt() と getInput() の定義を加えるのみとし，以下に示している部分に
関しては，一切の変更・追記は認めない．
以下の部分が変更・追記されたプログラムは正しく動作しても 0 点とする．

#include<stdio.h>
#include<ctype.h>   // 数字の判定に利用
#include<string.h>  // 文字列長の取得に利用

#define BUFSIZE 128

void getInput(char msg[], char n[]);
int isInt(char n[]);

int main(){
    char buf[BUFSIZE];
	
    getInput("整数値を入力してください：", buf);
    if(isInt(buf))
        printf("%s は非負の整数値です\n", buf);
    else
        printf("%s は非負の整数値ではありません\n", buf);
	
    return 0;
}


実行例1（赤文字がキーボードからの入力）：

$ ./ex8_3
整数値を入力してください：123
123 は非負の整数値です


実行例2（赤文字がキーボードからの入力）：

$ ./ex8_3
整数値を入力してください：0
0 は非負の整数値です


実行例3（赤文字がキーボードからの入力）：

$ ./ex8_3
整数値を入力してください：-12
-12 は非負の整数値ではありません

12x3 は非負の整数値ではありません
