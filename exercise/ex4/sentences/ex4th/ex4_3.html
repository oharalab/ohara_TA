実習課題4_3

【縦方向ヒストグラムの表示】

演習4-16では，0〜9までの100個の整数値を乱数を用いて生成し，各整数値の出現頻度を横方向に
のびるヒストグラムで可視化した．
ここでは，演習4-16のプログラム p4_16.c を参考に，以下の実行例のように垂直方向にのびるヒストグラムを
ターミナルウィンドウ上に描画することを考える．
しかし，ターミナルウィンドウ上では左から右へ行ごとにしか出力できない．
そのため，ヒストグラムを表示させる縦 max_size，横 bin_size の矩形領域を想定し，
ヒストグラムが描画される座標には 1，そうでない座標には 0 を保持する2次元配列 hist[][] をまず作成し，
その情報をもとにヒストグラムを描画するものとする．
そのような考えの下，2次元配列 hist[][] に値を設定する関数 make_hist()，hist[][] の情報に従って
実際に縦方向のヒストグラムを描画する関数 print_vhist() を実装し，以下のプログラム ex4_3.c を完成させよ．
なお，関数 get_data() は p4_16.c と同じものを実装せよ． 

/*
 * ex4_3.c
 */
#include 
#include 

#define bin_size 10
#define max_size 20
#define d_size 100

void get_data(int d[]);
void make_hist(int d[], int h[][max_size]);
void print_vhist(int h[][max_size]);

int main(){
    int data[d_size] = {};
    int hist[bin_size][max_size] = {};
	
    get_data(data);
    make_hist(data, hist);
    print_vhist(hist);

    return 0;
}
