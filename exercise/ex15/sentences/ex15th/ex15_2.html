実習課題15_2

【最小値・最大値】

引数で与えられた整数型配列の値の中から最小値を見つける関数 min()，および，最大値を
見つける関数 max() を実装し，以下のプログラム ex15_2.c を完成させよ．
なお，ex15_2.c の以下の変更は認めない．


#include<stdio.h>

int min(int *d, int num);
int max(int *d, int num);

int main(){
    int data[] = {4, 5, 1, 2, 7, 10, 8};
	
    printf("最小値：%d\n", min(data, (int)sizeof(data)/sizeof(int)));
    printf("最大値：%d\n", max(data, (int)sizeof(data)/sizeof(int)));
	
    return 0;
}


実行結果例（赤文字がキーボードからの入力）：
$ ./ex15_2.exe
最小値：1
最大値：10
