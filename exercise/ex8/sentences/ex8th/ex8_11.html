【西暦・月の入力】

実習課題8_10 作成した ex8_10.c に，月の正しい入力を求める関数 getMonth()
を実装し，以下のプログラム ex8_11.c を完成させよ．
なお，ex8_11.c は以下の部分に各関数の定義を加えるのみとし，以下に示している部分に
関しては，一切の変更・追記は認めない．
以下の部分が変更・追記されたプログラムは正しく動作しても 0 点とする．

#include<stdio.h>
#include<ctype.h>
#include<string.h>
#include<stdlib.h>

#define BUFSIZE 128

void getInput(char msg[], char n[]);
int isInt(char n[]);
int getYear(void);
int getMonth(void);

int main(){
	int year, month;
	
	year = getYear();
	printf("西暦：%d\n", year);
	month = getMonth();
	printf("月：%d\n", month);

	return 0;
}
